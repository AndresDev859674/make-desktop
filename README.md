# make-desktop
A CLI script you can use doesn't waste your time simply creating a .desktop file for application desktop entries

## Why i created this?
Every time I open an application or other executable file, I want it to be in the KDE start menu. I know it's through the .desktop folder in `~/.local/share/applications/`, but I'm wasting time just adding it there. So I made this program to save my time, and yours.

# installing
Download the bash script (./make-desktop.sh)

Use Chmod to convert to executable : chmod +x ./make-desktop.sh

and if you wanna help use the `./make-desktop.sh -h`
